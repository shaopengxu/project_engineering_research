# Bookmark API — 模板填写示例

本目录是 small-project 开发流程模板的**完整填写示例**，展示各模板文件在 HTTP REST API 类型项目中的预期填写效果。

## 项目简介

Bookmark API 是一个个人书签收藏管理的 REST API 服务（TypeScript + Express + SQLite），包含两个业务模块（书签管理、标签管理）和三个技术模块（bookmark、tag、shared/infra），符合 small-project 的适用范围。
