# Todo CLI — 模板填写示例

本目录是 small-project 开发流程模板的**完整填写示例**，展示各模板文件在实际项目中的预期填写效果。

## 项目简介

Todo CLI 是一个命令行 Todo 管理工具（Node.js + TypeScript），包含两个业务模块（任务管理、数据持久化）和三个技术模块（cli、task、storage），符合 small-project 的适用范围。
