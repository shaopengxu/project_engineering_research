# {项目名称} 接口契约

> 每个命令都必须包含四要素：**输入（参数+选项）、输出（stdout/stderr）、业务规则、错误处理**。

## 通用约定

### 全局选项
| 选项 | 缩写 | 类型 | 默认值 | 说明 |
|------|------|------|--------|------|
| --help | -h | boolean | false | 显示帮助信息 |
| --version | -V | boolean | false | 显示版本号 |
| --verbose | -v | boolean | false | 输出详细日志 |
| --config | -c | string | ./{tool}rc | 指定配置文件路径 |

### 退出码
| 退出码 | 含义 |
|--------|------|
| 0 | 成功 |
| 1 | 一般错误 |
| 2 | 参数错误 |

### 输出约定
- 正常结果输出到 stdout
- 错误和日志输出到 stderr
- 支持 `--json` 时，stdout 输出 JSON 格式（便于管道）
